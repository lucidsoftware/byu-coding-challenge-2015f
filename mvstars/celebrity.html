Description

You're hoping to be a news reporter one day and you're looking for a movie star to interview. Your friend has given you a method for determining if there is a celebrity in the room, and who it is.
He tells you that a person is a celebrity if everyone else in the room knows them, and they don't know anybody in the room. Find whether or not there is a celebrity in the room, and if there is who it is.

Input

The first line will contain two numbers 
n (2 <= n <= 1,000)
, the number of people in the room, and 
e (0 <= e <= n*(n-1))
the number of relationships. We will reference people by numbers between 
0
and 
n-1
.
The next 
e
 lines will each contain two numbers 
a
, and 
b
, which means that person 
a
 knows person 
b
.

Output

If there is no celebrity, print 
-1
, otherwise print the celebrity's number.

Examples

Input Input 
4 6
1 2
0 2
0 3
2 3
2 0
1 3

4 6
1 2
0 2
0 3
2 3
1 3
3 1
Output Output 
3
 
-1
