Description

In order to represent a 3 dimensional cube in 2 dimensions it needs to be unfolded. There are many ways to unfold a cube, to make this easier we always unfold the cube in the same way 

       --------       
       |      |       
       |  1   |       
       |      |       
----------------------
|      |      |      |
|  2   |  3   |  4   |
|      |      |      |
----------------------
       |      |       
       |  5   |       
       |      |       
       --------       
       |      |       
       |  6   |       
       |      |       
       --------
We don't know what orientation the cube was before it was unfolded. All we have is the 2d representation of the cubes and we want to know if 2 cubes are the same in 3d after refolding and rotations.
When refolding an unfolded cube we can either fold the edges up or fold the edges down, since we don't know how it was unfolded two cubes are the same if one is refolded up and the other is refolded down. For example, if you fold the edges up, face 
3
would be on the bottom, but if you folded the edges down 
3
would be on the top.

Input

The first line will be a number 
n
. 
n
testcases will follow. Each test case consists of two flatten cubes separated by an empty line.
You will receive two flattened cubes separated by an empty line.
Each face of the cube will be represented with a single lowercase letter. A cube can have multiple identical faces.

Output

Output 
same
if they are the same, and 
different
if they are different.
